# 🚀 Featured Projects

## 💻 Distributed CodeForge
Cloud-native, agentic collaborative coding workspace built with **Spring Boot, Spring Cloud, Kafka, Kubernetes, React, and Spring AI.**

- 6-microservice control plane across network-isolated K8s namespaces for secure runtime sandboxing
- Cut sandbox preview spin-up latency **90%** (15s → <1.5s) via a pre-warmed K8s runner pod pool + Redis route caching
- Self-correcting agentic feedback loop using Spring AI tool calling to run builds and auto-resolve compiler errors via pgvector error-fix memories
- 5 path-filtered GitHub Actions workflows with Jib (daemonless builds) + keyless GCP Workload Identity Federation
- Stripe-powered subscription billing with token usage quotas and locked-down sandbox network egress

🔗 GitHub | Live Demo

## 🏨 StayNest – Hotel Booking Platform
Hospitality reservation REST API built with **Java, Spring Boot, PostgreSQL, Stripe, and Spring AI.**

- 90+ source files, 25+ tests, clean layered architecture with a Decorator-pattern pricing engine
- Signature-verified Stripe checkout + transactional refund handlers with DB-first state persistence
- Pessimistic locking on room inventory to eliminate double-booking, plus a scheduled job expiring unpaid reservations every 5 minutes
- 3 Spring AI integrations: dynamic pricing, semantic property search, and a review Q&A chatbot (pgvector-backed), each with fallback handlers
- HttpOnly JWT cookies, BCrypt hashing, RBAC, and token versioning for global sign-out on password reset

🔗 GitHub | Live Demo
